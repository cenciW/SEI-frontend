# Modelos 3D para Visualização de Culturas

## Instruções

Coloque seus modelos 3D nesta pasta com os seguintes nomes:

- `cannabis.glb` ou `cannabis.gltf` - Modelo da planta de Cannabis
- `corn.glb` ou `corn.gltf` - Modelo da planta de Milho
- `tomato.glb` ou `tomato.gltf` - Modelo da planta de Tomate
- `wheat.glb` ou `wheat.gltf` - Modelo da planta de Trigo
- `lettuce.glb` ou `lettuce.gltf` - Modelo da planta de Alface

## Formatos Suportados

- **GLB** (recomendado): Formato binário, arquivo único
- **GLTF**: Formato JSON com arquivos separados

## Onde Encontrar Modelos 3D

1. **Sketchfab**: https://sketchfab.com (muitos modelos gratuitos)
2. **TurboSquid**: https://www.turbosquid.com
3. **CGTrader**: https://www.cgtrader.com
4. **Free3D**: https://free3d.com

## Notas

- O sistema usa **fallback** para geometria procedural se o modelo não for encontrado
- Modelos devem estar otimizados (baixo poly count) para performance
- Tamanho recomendado: < 5MB por modelo
